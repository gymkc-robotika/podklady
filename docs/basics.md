Základy
=======

Struktura (tok) programu
------------------------

- Větvení (`if`, `if` / `else`)
- Smyčky (`for`, `while`, `do` / `while`)
- Funkce
 - parametry
 - návratové hodnoty

Data
----

- Proměnné
- Konstanty

Mohou být různého typu

- primitivní (základní)
  - čísla
  - řetězce
  - odkazy (ukazatele, reference)
- pole
  indexovatelné, např: `pole[0] = 123`, `pole[1] = pole[0] * 2`
- složené
  - třídy / struktury
